### Сценарий командной строки 

-[]

1. Опеределение 
```
Сценарии командной строки — это наборы тех же самых команд, которые можно вводить с клавиатуры, собранные в файлы и объединённые некоей общей целью. 
При этом результаты работы команд могут представлять либо самостоятельную ценность, либо служить входными данными для других команд. 
Сценарии — это мощный способ автоматизации часто выполняемых действий.
```


2. Выполнение несколько команд 
```
Если говорить о командной строке, она позволяет выполнить несколько команд за один раз, введя их через точку с запятой:

pwd       : present working directory ( выводит на экран сведения о текущей рабочей директории )
whoami    : who am i ?                ( показывает данные о пользователе, под которым вы вошли в систему )

yao@pc-gaming:~$ pwd
/home/yao
yao@pc-gaming:~$ pwd; whoami
/home/yao
yao
yao@pc-gaming:~$ 

```


3. ARG_MAX 
```
Используя подобный подход, вы можете совмещать сколько угодно команд в одной строке, 
ограничение — лишь в максимальном количестве аргументов, которое можно передать программе. 
Определить это ограничение можно с помощью такой команды:

$ getconf ARG_MAX
506873
```


4. Создание bash script 
```
$ touch myscript.sh

$ cat myscript.sh 
#!/bin/bash
# This is a comment
pwd
whoami

$ chmod +x ./myscript.sh
```
