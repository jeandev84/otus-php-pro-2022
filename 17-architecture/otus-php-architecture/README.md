### Архитектура кода 

1. Example DDD
```
https://github.com/CodelyTV/php-ddd-example
Infrastructure: (Содерживает всё что с внешнием миром, всё что меняет часто)
  Controller : принимает HTTP запрос
  Repository : подключается к БД
  Command    : принимает входные данных через терминал
Application: (Описивает поряка выполнения чего то бизнес логика, средне меняется)
  Service
Domain: (Описивает бизнес-сущности чего-то, всё что меняется менее редко )
  Model или Entity ( Model - это например товар, физ. лицо, паспортные данные )
```
